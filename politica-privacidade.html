Política de Privacidade

Última atualização: maio de 2026 · RMEL Soluções de Tecnologia Ltda

1. Quem somos

A RMEL Soluções de Tecnologia Ltda (CNPJ 63.785.682/0001-05) é a operadora do sistema Júlia — plataforma de atendimento automatizado via WhatsApp para clínicas de saúde e estética. Para dúvidas sobre privacidade, entre em contato pelo e-mail guerra.raffa@gmail.com.

2. Dados coletados

Coletamos dados em duas situações distintas:

a) Formulário de contato neste site

  • Nome e nome da clínica
  • E-mail
  • Número de WhatsApp (opcional)
  • Mensagem enviada

b) Pacientes atendidos pela Júlia (via WhatsApp das clínicas clientes)

  • Número de telefone WhatsApp
  • Nome (fornecido pelo próprio paciente)
  • Histórico de conversas com o bot
  • Data, horário e serviço agendado

3. Finalidade do tratamento

  • Responder mensagens enviadas pelo formulário de contato
  • Operar o sistema de agendamento automatizado para as clínicas contratantes
  • Enviar lembretes de agendamento aos pacientes
  • Melhorar a qualidade do atendimento automático

4. Base legal (LGPD)

O tratamento dos dados é realizado com base no legítimo interesse (Art. 7º, IX — execução do contrato ou procedimentos preliminares) e no consentimento do titular ao enviar o formulário ou interagir com o bot.

5. Compartilhamento de dados

Os dados não são vendidos ou compartilhados com terceiros para fins comerciais. Utilizamos apenas os seguintes prestadores de serviço essenciais à operação:

  • Resend — envio de e-mails de notificação (dados necessários: e-mail do destinatário)
  • Cloudflare — infraestrutura do site e workers (dados de acesso em logs)
  • MEGA-API — gateway WhatsApp (mensagens transmitidas em trânsito)
  • OpenAI — processamento de linguagem natural para o bot (conteúdo das mensagens)

Todos esses parceiros possuem políticas de privacidade próprias e tratam os dados apenas conforme necessário para a prestação do serviço.

6. Armazenamento e segurança

Os dados são armazenados em servidores localizados nos Estados Unidos (Hetzner Cloud — Ashburn, VA) com acesso restrito por autenticação. Os dados de cada clínica são isolados entre si (arquitetura multi-tenant). Utilizamos conexões HTTPS/TLS em todos os pontos de acesso.

7. Direitos do titular

Nos termos da LGPD (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar os dados que possuímos sobre você
  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a anonimização, bloqueio ou eliminação dos seus dados
  • Revogar o consentimento a qualquer momento

Para exercer qualquer desses direitos, envie um e-mail para guerra.raffa@gmail.com com o assunto "LGPD — [seu direito]".

8. Retenção dos dados

Os dados do formulário de contato são mantidos pelo tempo necessário para responder à sua solicitação e, por até 12 meses, para fins de registro. Os dados dos pacientes atendidos pela Júlia são mantidos pelo tempo definido no contrato com a clínica contratante, respeitando obrigações legais aplicáveis.

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data de "última atualização" no topo da página indica a versão vigente. Mudanças relevantes serão comunicadas pelo e-mail cadastrado, quando aplicável.
